Splunk Dashboard Creation

Using SPL Queries

Below is the SPL queries i used to create the dashboard. This dashboard is using the "Network Intrusion Detection" dataset from kaggle.com. I created a dashboard that would monitor and count vulnerabilities such as udp traffic and port 80. Adding this dashboard to my home screen in splunk to monitor would have a faster response time. Query:

source="hello_1.csv" "80" | eval Heading="Port 80 Traffic" | append [ | search "udp" | eval Heading="UDP Traffic" ] | stats count as Count by Heading | eventstats sum(Count) as Total | eval Count_Percentage=(Count*100/Total) | table Heading Count Count_Percentage
